# Factory Scope

Factory Scope は、日本の上場企業の工場・設備投資・生産能力・立地情報を可視化する、投資家向けのリサーチアプリケーションです。

## コンセプト

一般的な株式投資ツールは、株価、決算、PER、PBR、チャート、ニュースを中心に企業を分析します。

Factory Scope は、それとは別の観点を扱います。

> その会社はどこに、何を作るために、いくら投資していて、いつ業績に効いてくるのか。

工場や設備投資の情報を整理することで、決算数値だけでは見えにくい成長投資、供給能力、立地リスク、テーマ性を調べやすくします。

## 想定ユーザー

- 個人投資家
- 半導体・電子部品株を調べる投資家
- 設備投資を重視する投資家
- サプライチェーンを調べたい人
- 日本の製造業を企業単位・工場単位で分析したい人

## 初期MVPの対象

最初のバージョンでは、半導体・電子部品関連の日本企業を中心に扱います。

例:

- キオクシア
- イビデン
- 新光電気工業
- TOWA

- 村田製作所

- ルネサスエレクトロニクス

## 主な機能

- 上場企業検索
- 工場・事業所データベース
- 設備投資案件データベース
- 工場詳細ページ
- 設備投資タイムライン
- 出典付きデータ管理
- AI、半導体、HBM、EV、電池、データセンターなどのテーマタグ
- 稼働状況の管理
- 抽出データの信頼度スコア
- 手動確認済みフラグ

## 重視する情報

- 工場名・事業所名
- 所在地
- 土地面積
- 建物面積
- 生産品目
- 生産能力
- 設備投資額
- 稼働予定時期
- 量産開始時期
- 関連テーマ
- 出典資料
- 出典テキスト
- データの信頼度

- Company: 企業
- Facility: 工場・事業所
- CapexProject: 設備投資案件

- SourceReference: 出典情報
- ThemeTag: テーマタグ

## 開発方針

- Issue駆動で開発する
- 1 Issue = 1 Pull Request を基本にする
- 出典を必ず保存する
- 自動抽出データは信頼度スコアを持たせる
- 手動確認前のデータは確定情報として扱わない
- 最初から完全自動化を狙わず、手動登録と出典管理を優先する

## 投資助言ではないこと

Factory Scope は調査・情報整理を目的としたアプリケーションです。
投資判断そのものを推奨するものではありません。
